# 平台界面截图说明

## 登录页面

![登录页面](../images/login-screenshot.png)

登录页面包含用户名输入框、密码输入框和登录按钮。支持记住密码和找回密码功能。

## 用户工作区

![用户工作区](../images/workspace.png)

用户登录后进入工作区，左侧为知识库列表，右侧为文档目录和预览区域。

## 管理中心

![管理中心](../images/admin-dashboard.png)

管理员可以通过顶部导航栏进入管理中心，查看系统概览、用户管理、角色管理等功能。

## 文档预览

![文档预览](../images/document-preview.png)

文档预览支持 Markdown 渲染、表格展示、代码高亮和图片显示。所有内容经过安全清洗处理。

## 引用定位

![引用定位](../images/citation.png)

问答结果中的引用可以定位到原始文档的对应位置，包含页码、片段和相关度分数。
