# 员工差旅报销管理办法（节选）

本办法适用于公司全体正式员工因公出差产生的交通、住宿与餐饮费用报销。出差前须在 OA 提交出差申请，经直属主管与部门负责人审批通过后方可出行；紧急出差可先电话报备，并在出发后一个工作日内补齐申请。

交通费用：同城出差优先地铁与公交；跨城出差经济舱机票或二等座高铁可实报实销，商务舱与一等座须事前书面特批。打车仅限无公共交通、加班至二十二点后或携带大件样品等情形，并保留行程单。

住宿标准：一线城市（北京、上海、广州、深圳）每晚不超过六百元，其他城市不超过四百五十元；超标部分由个人承担。两人同行原则上合住，性别不同或职级差异可分开预订。

餐饮补贴按自然日计发：出差当日有早餐服务的酒店，当日餐饮补贴减半；全天出差一线城市一百二十元、其他城市一百元。招待客户产生的餐费不占用个人补贴额度，但须单独填写业务招待单并附菜单与参与人名单。

报销时限为返回公司后十个工作日，逾期未提交者须书面说明原因；无正当理由超过三十日的，财务可拒绝受理。发票须真实合法，抬头与税号齐全。财务审核发现虚假票据的，除追回款项外，按公司纪律处分相关制度处理。

本办法由财务部负责解释，自发布之日起施行，原《差旅临时规定》同时废止。
