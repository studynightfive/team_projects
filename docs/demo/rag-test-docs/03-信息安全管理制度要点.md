# 信息安全管理制度要点

为保护公司商业秘密与用户数据，全体员工须遵守本制度。信息系统账号实行实名制，禁止共享个人登录凭证；临时外包人员使用专用账号，项目结束后五个工作日内由信息安全岗禁用。

数据分级：公开、内部、敏感、机密四级。敏感及以上数据不得存入个人网盘或私人即时通讯；外发须走数据防泄漏审批，并记录接收方与用途。笔记本电脑离开工位须锁屏，屏幕保护等待时间不超过五分钟。

办公网络安全：仅允许接入公司认证 Wi-Fi 或经批准的 VPN；禁止私自架设无线热点桥接内网。发现钓鱼邮件应立即上报安全运营中心，不得点击可疑链接或打开未知附件。

漏洞与补丁：终端操作系统与办公软件须开启自动更新；高危漏洞补丁应在发布后七日内完成安装，服务器由运维按变更窗口统一处理。未经授权不得扫描内网或进行渗透测试。

事件响应：发生疑似数据泄露、勒索或账号被盗，当事人须在三十分钟内通知直属主管与安全运营中心，并保留现场证据。隐瞒不报造成扩大损失的，按严重违纪处理。

本制度由信息安全委员会解释，每年至少评估一次；与国家法律法规冲突时，以法律法规为准。
