# 客户投诉处理流程

客户投诉处理遵循「快速响应、闭环跟踪、原因改进」原则，适用于电话、邮件、工单与应用内反馈渠道。一线客服须在受理后十五分钟内完成首响，告知客户工单编号与预计处理时限。

分级标准：一般投诉指功能咨询、体验建议与可即时解决的操作问题，二十四小时内关闭；重要投诉指影响核心业务可用性或涉及费用争议，四小时内升级产品或财务值班；重大投诉指群体性舆情、数据安全或监管问询，一小时内通知客服负责人与分管副总，并启动专项应急小组。

处理要求：每个工单须记录问题现象、影响范围、临时方案与最终方案；涉及退款的，须核对订单号、支付流水与合同条款后由财务复核。禁止向客户承诺超出制度授权的补偿。

升级路径：一线无法解决时，按业务线升级二线；二线超时未响应，系统自动抄送部门负责人。重大投诉结案后五个工作日内须输出改进措施，并纳入双周质量例会跟踪。

客户回访：重要与重大投诉结案后四十八小时内完成一次回访，确认问题是否复发与满意度。回访不满意的，重新打开工单并标注「二次处理」。

本流程由客户成功部负责解释，与《服务质量考核办法》配套执行。
