## 文档 1：差旅报销管理办法

| 问题 | 参考答案要点 |
|------|----------------|
| 一线城市差旅住宿标准是多少？ | 北京、上海、广州、深圳每晚不超过 **600 元** |
| 出差餐饮补贴怎么算？ | 一线城市全天 **120 元**，其他城市 **100 元**；酒店含早餐则当日补贴减半 |
| 报销要在返回后多久提交？ | **10 个工作日**；无正当理由超过 30 日财务可拒绝受理 |
| 什么情况下可以打车报销？ | 无公共交通、加班至 **22 点**后，或携带大件样品等，并保留行程单 |

## 文档 2：产品上线发布检查清单

| 问题 | 参考答案要点 |
|------|----------------|
| 上线前多久要完成清单勾选？ | 上线窗口开始前 **两小时** |
| 什么情况下要暂停上线并评估回滚？ | 错误率超过基线 **两倍**，或核心接口可用性低于 **99.5%** |
| 上线后多久要持续关注？ | 上线后 **24 小时**内；并完成《上线复盘》才能关闭变更单 |
| 发布前数据库迁移有什么要求？ | 须在预发演练成功且 **可回滚** |

## 文档 3：信息安全管理制度要点

| 问题 | 参考答案要点 |
|------|----------------|
| 临时外包人员账号何时禁用？ | 项目结束后 **5 个工作日**内由信息安全岗禁用 |
| 敏感数据能否存个人网盘？ | **不可以**；外发须走数据防泄漏审批 |
| 高危漏洞补丁时限？ | 发布后 **7 日内**完成安装 |
| 疑似数据泄露要多久上报？ | **30 分钟内**通知直属主管与安全运营中心 |

## 文档 4：客户投诉处理流程

| 问题 | 参考答案要点 |
|------|----------------|
| 一线客服首响时限是多少？ | 受理后 **15 分钟**内完成首响 |
| 一般投诉与重要投诉的关闭/升级时限？ | 一般 **24 小时**关闭；重要 **4 小时**内升级 |
| 重大投诉要多久通知管理层？ | **1 小时**内通知客服负责人与分管副总 |
| 重要投诉结案后回访要求？ | **48 小时**内完成一次回访；不满意则重新打开工单 |
