# M02–M14 本地页面浏览器证据

本目录保存 M02–M14 后续业务页面的确定性浏览器回归证据。它验证页面结构、响应式、导航可达性和本地安全边界，不代表真实认证、权限或业务接口已经联调。

## 覆盖结果

- 16 个后续业务路由分别覆盖 1440px、1280px 和 375px：`16 × 3 = 48` 个场景。
- 另覆盖样例文档详情页和管理员用户页的 1440px 侧栏折叠态：2 个场景。
- 合计 50 个场景，50/50 通过。
- 浏览器控制台错误为 0，业务 `/api` 请求为 0。
- 机器可读结果见 [`browser-report.json`](browser-report.json)，对应的精确源码提交以该报告顶层 `sourceCommit` 字段为准，本文不重复硬编码提交号。

## 本地边界

- 页面使用 M01 V2 正式基线和固定 design-only 数据。
- 筛选、弹窗、抽屉、确认和状态切换只改变本地组件状态。
- 报告中的业务 `/api` 请求为 0；未验证真实登录、Cookie、权限码、上传、鉴权下载、轮询或流式问答。
- 因此该证据只支持“M02–M14 本地页面已开发”，不能支持“业务里程碑联调完成”或“统一前端 P0 最终交付完成”。

## 复现命令

先在仓库根目录启动本地开发服务器，再执行浏览器验收：

```powershell
pnpm.cmd run dev:web
pnpm.cmd run verify:web:browser
```
