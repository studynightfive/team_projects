# 统一 Web 前端功能说明

> 更新时间：2026-07-17
>
> 状态：历史设计快照。当前真实联调范围、权限和接口以 `README.md`、`docs/api/openapi.yaml` 与运行时实现为准。
>
> 适用范围：`frontend/web/` 统一 Vue 3 前端
>
> 面向读者：产品负责人、测试人员、前端与后端开发人员

## 1. 当前结论

当前前端已经形成一套可直接浏览和操作的本地产品原型，包含普通用户工作区、企业 AI 搜索、知识库、智能问答、个人资产、管理中心、登录页和异常页。

需要特别区分以下三种状态：

- **本地交互已实现**：按钮、筛选、标签页、抽屉、确认框等可以操作，但数据通常只保留在当前页面，刷新后恢复。
- **仅界面提示**：点击后会说明后续能力，但不会执行真实上传、下载、同步、登录或管理操作。
- **等待后端联调**：真实数据、认证、权限和持久化必须等待 OpenAPI 契约与后端接口，不应把当前固定数据当作生产结果。

在仓库根目录启动：

```powershell
pnpm.cmd run dev:web
```

本地入口：

- 用户工作区：<http://127.0.0.1:5173/>
- 管理中心：<http://127.0.0.1:5173/admin>
- 登录页：<http://127.0.0.1:5173/login>

## 3. 公共布局与导航功能

### 3.1 桌面侧栏

- 展示当前工作区的核心功能入口，并高亮当前页面。
- 支持展开和折叠；在 768–1180px 的平板宽度下默认折叠。
- “用户与角色”“文档与任务”在管理中心显示对应子导航。
- 用户工作区与管理中心可相互切换。
- 底部账号区域可以打开个人资料、搜索设置以及其他账号操作入口。
- 搜索设置不在用户主导航中展示，继续通过底部账号菜单进入 `/settings`。
- 用户主导航前两项依次为“AI 搜索”“AI 助手”；移动端底部四个高频入口顺序不变。

个人中心会打开紧凑的固定演示资料弹窗，展示身份、当前工作区、认证接入状态和资料来源；桌面使用双列概览，移动端自动改为单列，并可直接进入偏好设置。弹窗不读取真实账号资料，系统设置和退出登录仍只显示后续功能说明。

### 3.2 顶部栏

用户工作区：

- 面包屑和页面名称随路由变化。
- 全局搜索支持输入关键词后按 `Enter` 进入 `/search`。
- `Ctrl + K` 或 `Command + K` 可快速聚焦全局搜索框。
- 数据源状态入口展示当前模拟数据源的可用数量，并跳转 `/data-sources`。
- 通知和帮助按钮分别进入 `/notifications` 与 `/help`；桌面端鼠标移入或键盘聚焦通知铃铛时展示最近 4 条通知，未读数量与当前会话内的已读操作同步。
- 提供管理中心入口和个人资料入口。

管理中心：

- 显示当前管理页面名称。
- 可切换“演示/生产”环境标签，但不会改变真实运行环境。
- 通知按钮进入 `/admin/notifications`，桌面端复用同一通知预览交互；管理通知与用户通知使用独立的当前会话状态。
- 提供返回用户工作区和个人资料入口。

### 3.3 移动端导航

- 767px 以下使用顶部菜单、完整导航抽屉和底部快捷导航。
- 用户端底部保留搜索、知识库、搜索历史和 AI 助手四个高频入口。
- 管理端底部保留概览、用户与角色、知识库和审计四个高频入口。
- 用户端通知和帮助入口保留在移动顶栏；移动端不显示悬停预览，通知铃铛直接进入通知中心；偏好设置可从个人资料弹窗进入。
- 导航抽屉支持遮罩关闭、`Escape` 关闭、键盘焦点约束和关闭后焦点返回。

## 4. AI 搜索工作台

### 4.1 AI 搜索首页 `/`

用途：让用户不用先理解知识库目录，直接用自然语言搜索企业知识。

主要功能：

1. **自然语言输入**
   - 支持多行问题输入和自动增高。
   - `Enter` 提交搜索，`Shift + Enter` 换行。
   - 支持一键清空当前问题。

2. **四种搜索模式**
   - 智能搜索：综合多个数据源生成带引用的答案。
   - 精确检索：优先查看原始文档和高匹配内容。
   - 深度研究：用于复杂问题的多步骤检索和交叉验证。
   - 文档问答：限定在选择或添加的文档内回答。

3. **搜索范围与模型**
   - 可选择企业知识库、项目文档、规章制度、会议记录、业务数据、个人文件和互联网信息。
   - 至少保留一个搜索范围，避免提交无来源查询。
   - 可在企业通用模型、长文档分析模型和快速检索模型之间切换。

4. **附件选择与校验**
   - 支持 PDF、Word、TXT、Markdown 和 Excel 文件。
   - 单个文件不超过 10 MB，最多添加 5 个。
   - 可查看和移除已选择附件。
   - 当前只校验并传递文件名，不读取或上传文件内容。

5. **快捷提问和任务入口**
   - 常用问题点击后只回填到输入框，不会自动提交。
   - 快捷开始包含上传文档、创建深度研究、进入知识空间、查看收藏和管理数据源。

6. **工作延续信息**
   - 最近搜索可以直接重新打开对应问题。
   - 知识空间展示常用部门空间，并可进入空间详情。
   - 数据源概览展示连接状态和已同步内容规模。

7. **搜索上下文**
   - 展示当前问题、搜索范围、附件、模型、可用数据源和异常数据源。
   - 1440px 及以上显示为右侧栏，较窄屏幕显示为可关闭抽屉。

当前边界：提交后调用的是本地模拟服务，不会上传附件、调用模型或请求真实企业数据。

### 4.2 AI 搜索结果 `/search`

用途：在同一页面同时提供可阅读的 AI 结论和可核验的原始来源。

主要功能：

1. **查询摘要与重新搜索**
   - 展示当前问题、模式、范围、数据源数量和模拟状态。
   - 可直接修改问题、模式、范围、模型和附件后再次搜索。
   - 支持重新生成本地答案。

2. **结果操作**
   - 复制答案到剪贴板；浏览器拒绝权限时给出明确提示。
   - 将当前答案导出为本地 Markdown 文件。
   - 收藏或取消收藏当前答案；状态只保留在本页。

3. **AI 答案标签页**
   - 展示摘要、结构化 Markdown、表格、结论依据和风险提示。
   - Markdown 禁用原始 HTML，并经过 DOMPurify 二次过滤。
   - 正文引用编号和来源列表都可以打开对应文档预览。
   - 支持“有帮助、没有帮助、答案不准确、来源不可靠、内容已过期”等反馈。
   - “继续了解”中的相关问题可直接发起下一次搜索。

4. **原始结果标签页**
   - 支持结果内关键词搜索。
   - 支持按内容类型、数据源、部门、更新时间筛选。
   - 支持按相关度或更新时间排序，并切换宽松/紧凑密度。
   - 展示来源、文件类型、命中关键词、负责人、验证状态、权限状态和模拟相关度。
   - 单条结果可以本地收藏或打开文档预览。
   - 移动端高级筛选以底部抽屉呈现。

5. **文档预览抽屉**
   - 展示文档类型、空间、负责人、更新时间、验证状态和权限状态。
   - 支持上一个/下一个命中位置，并滚动到当前命中内容。
   - 支持复制包含文档 ID 的本地占位链接和本地收藏；当前重新打开该链接不会自动恢复文档预览。
   - “在新页面打开”当前只说明等待文档接口，不会访问不存在的原文地址。

6. **搜索上下文**
   - 展示问题、范围、附件、模型、数据源统计和全部引用来源。
   - 点击引用可打开同一个文档预览抽屉。

7. **页面状态**
   - 覆盖检索中、完成、部分结果和失败状态。
   - 失败后保留当前问题并提供重新搜索入口。

当前边界：答案、引用和原始结果来自确定性 Mock。固定差旅问题会展示完整差旅样例，其他问题只返回与当前问题对应的模拟检索说明；原始结果也不是针对问题执行的真实检索。真实检索耗时、权限过滤、模型生成和流式响应尚未接入。

### 4.3 深度研究 `/research`

用途：把复杂问题拆成多个可检查步骤，帮助用户识别证据边界，而不是直接给出无法核验的结论。

主要功能：

- 展示研究问题、当前状态和研究摘要。
- 展示六阶段计划：理解问题、制定计划、检索企业知识、检索外部信息、分析交叉验证、生成结论。
- 每一步显示等待、执行、完成或失败状态。
- 展示当前连接的引用来源、研究范围和中间结论。
- 未全部完成时显示最终报告生成进度。
- “导出报告”在研究未完成时只显示说明，不生成文件。

当前边界：页面使用一条固定模拟研究任务，不启动真实异步研究、外部搜索或报告任务。

## 5. 知识与文档功能

### 5.1 企业知识库 `/knowledge`

用途：按组织维护的知识库浏览企业文档。

主要功能：

- 按名称或说明搜索知识库。
- 按知识库类型筛选。
- 卡片展示知识库类型、简介、文档数量和更新时间。
- 可进入对应文档目录。
- 提供进入搜索结果页的快捷入口。
- 无匹配项时显示空状态。

当前边界：知识库列表和可见范围是固定数据，尚未依据真实用户权限加载。

### 5.2 文档目录 `/knowledge/:kb_id`

用途：在指定知识库内筛选、选择和预览文档。

主要功能：

- 按文档名称、分类或负责人搜索。
- 按已索引、处理中、需复核等状态筛选。
- 支持单选和全选当前筛选结果。
- 展示分类、状态、页数、负责人和更新时间。
- 可进入文档预览页。
- 可预览“导出所选”操作及选中数量。
- 知识库不存在或无匹配文档时显示相应状态。

当前边界：“导出所选”不会创建导出任务，分页和权限字段等待接口契约。

### 5.3 文档预览 `/knowledge/:kb_id/documents/:document_id`

用途：检查文档详情页的信息结构、正文安全呈现和引用定位方式。

主要功能：

- 展示文档名称、分类、页数和负责人。
- 固定样例文档支持按页码目录切换发布目标、发布前检查、回滚流程和验收证据。
- 正文样例覆盖普通段落、表格、代码块和引用高亮。
- 支持返回文档目录和预览导出操作。
- 父知识库不存在、文档不属于当前知识库或正文无样例时显示安全状态。

当前边界：该页面使用受控 Vue 模板，不解析真实 Markdown；非固定样例文档不会借用其他文档内容冒充正文。

### 5.4 我的空间 `/spaces`

用途：按部门和主题组织常用知识，并从空间上下文直接发起 AI 搜索。

主要功能：

- 按空间名称、简介、负责人和部门筛选六个模拟空间。
- 卡片展示权限类型、部门、负责人、成员数和文档数。
- 支持通过 `?space=` 直接选中指定空间。
- 空间详情包含空间概览、文档列表、成员列表、热门问题和空间设置五个标签页。
- 标签页支持左右方向键、`Home` 和 `End` 键切换。
- “向此空间提问”及热门问题会带着知识库范围进入 AI 搜索。

当前边界：文档清单、成员权限和空间设置尚未接入；页面不会修改真实空间或权限。

## 6. 个人搜索资产与辅助功能

### 6.1 收藏内容 `/favorites`

用途：集中管理收藏的 AI 答案、文档、知识空间和常用问题。

主要功能：

- 按标题、摘要、标签或备注搜索。
- 按收藏类型和标签筛选。
- 添加、移除标签并编辑使用备注。
- AI 答案和搜索问题可进入搜索页，知识空间可进入指定空间；文档收藏当前只显示等待文档接口的提示。
- 删除收藏前进行二次确认。
- 无匹配内容时显示空状态。

当前边界：标签、备注和删除只影响当前页面，刷新后恢复，不同步服务器。

### 6.2 历史会话 `/conversations`

用途：管理此前的智能问答会话并继续提问。

主要功能：

- 按标题或摘要搜索，并支持只看置顶会话。
- 新建问答或进入固定会话继续提问。
- 重命名会话，支持 `Enter` 保存、`Escape` 取消。
- 删除前进行二次确认。
- 展示更新时间和消息数量。

当前边界：重命名和删除只影响当前页；固定分页为第 1/1 页，没有真实会话列表接口。

### 6.3 搜索历史 `/history`

- 可按原问题、模式和范围再次搜索。
- 可重命名、收藏/取消收藏或删除单条记录。

当前边界：通知和已读状态只保留在当前应用会话，不会请求接口或写入浏览器存储；刷新应用后恢复固定样例。

## 7. 管理中心功能

### 7.1 平台总览 `/admin`

用途：让管理员快速了解平台规模、运行健康和待治理事项。

当前边界：全部指标、服务状态、治理项和日志为固定演示数据；导出、治理处理和翻页仅显示提示。

当前边界：不读取或展示已有密钥，连通性测试不访问网络；温度和最大输出当前不会写回本地模型配置，再次打开会恢复默认值，参数仍等待 OpenAPI 确认。

## 9. 通用交互、状态与安全说明

### 9.1 页面状态

- 搜索工作台和业务页面提供加载、空、错误和部分结果等状态；另有独立 `/403` 页面和来源权限状态展示。
- 删除、批量清理、启停、归档等高影响操作在本地页面中提供确认步骤。
- 表格在卡片内部横向滚动，避免整个页面在移动端横向溢出。

### 9.2 键盘与无障碍

- 页面保持单一主标题，表单控件具有标签或可访问名称。
- 移动抽屉、搜索上下文、文档预览和高级筛选支持 `Escape` 关闭及焦点返回。
- 搜索结果和空间标签页支持键盘方向键、`Home`、`End` 切换。
- 主要状态变化通过可见文字或 `aria-live` 表达。
- 继续支持清晰焦点环和 reduced motion。

### 9.3 当前安全边界

- Mock 开发模式不允许未注册业务请求回退到真实网络。
- 前端不会把 Token、密码、密钥或真实个人敏感数据写入 URL、日志或浏览器存储。
- 模型管理的密钥字段只用于界面预览，并在离开相关编辑状态时清空。
- AI 搜索 Markdown 禁用原始 HTML，并使用 DOMPurify 过滤脚本、事件属性和危险链接。
- 下载页面不构造可猜测的静态文件地址。
- 管理入口和管理路由尚未按权限码限制，当前可访问不代表获得真实管理权限。
