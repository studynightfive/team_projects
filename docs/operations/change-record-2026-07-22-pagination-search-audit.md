# 2026-07-22 列表分页与 AI 搜索核查记录

## 修正范围

1. 命中率测试的指标、逐题结果和运行记录采用“中文（English）”双语标签。
2. 为用户、角色、部门、模型、知识库、文档、任务、审计、通知、收藏、下载、帮助、知识空间、检索候选、逐题结果和测试运行记录接入统一分页。
3. 删除 AI 搜索首页固定常用问题及其无效回填逻辑和样式。
4. AI 搜索辅助信息按 `documentId` 去重引用；缺少 ID 时按来源名与标题组合去重；引用概览不再显示“待确认”。
