## 通用检查

- [x] 每页只有一个 `h1`，顶栏标题与页面一致。
- [x] 桌面侧栏和移动完整抽屉可以进入全部正式业务路由，底部导航保留四个高频入口。
- [x] 动态详情页和组合菜单保持正确父项高亮。
- [x] 页面使用 M01 正式 tokens、按钮、卡片、圆角、阴影和焦点样式。
- [x] 1440/1280/375 无根级横向溢出，表格只在卡片内部滚动；两个 1440px 侧栏折叠态抽查通过。
- [x] 375px 主要操作至少 44px，内容单列且无遮挡。
- [x] 弹窗/抽屉支持 Escape 或明确关闭，关闭后焦点可回到触发元素。
- [x] 无 emoji、敏感数据、真实密钥、Token、可猜测下载地址或内部堆栈。
- [x] 控制台错误 0，业务 `/api` 请求 0，localStorage/sessionStorage 不写入凭据。

## 页面专项

- [x] 知识库、文档、搜索结果和引用可以沿固定样例路由互相跳转。
- [x] 三种检索模式切换后保留筛选参数，空关键词不触发伪请求。
- [x] 问答停止/重试只改变本地展示状态，离页没有遗留请求或定时器。
- [x] 会话重命名与删除有确认反馈，刷新后恢复固定数据。
- [x] 下载失败、过期和完成状态可区分，下载按钮不生成真实地址。
- [x] 模型密钥输入默认空，关闭面板后清空，不写 Store/Storage/URL。
- [x] 文档和任务状态逐条独立，失败与人工复核清晰可见。
- [x] 审计详情使用掩码和 RFC 5737 示例 IP，只读且无敏感请求体。

## 证据目录

证据已保存到 `docs/verification/m02-m14-local-pages/`：16 个后续业务路由各覆盖 1440/1280/375 三个视口，并补充两个 1440px 侧栏折叠态，共 50 个场景，结果为 50/50 通过；控制台错误与业务 `/api` 请求均为 0。精确源码提交以机器报告顶层 `sourceCommit` 字段为准。本结论只表示本地页面完成，不表示真实接口、认证或权限联调完成。
