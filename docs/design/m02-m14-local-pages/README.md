## 交付边界

- 实现正式路由表中的 M02–M14 页面，不新增 `/profile` 或未批准详情路由。
- 页面使用确定性 design-only 数据，所有业务 `/api` 请求保持为 0。
- 搜索、筛选、分页、选中、确认框、抽屉和状态切换只在当前页面内生效，不显示成真实服务端保存成功。
- 真实登录、Cookie、权限码、路由守卫、SSE、上传、下载和轮询必须等待 OpenAPI 后另行联调。
- M02–M14 只有完成真实契约、接口、权限和测试后才能标记里程碑完成；本轮只标记“本地页面完成”。

## 本地入口

```powershell
pnpm.cmd install --frozen-lockfile
pnpm.cmd run dev:web
```

用户端从 http://127.0.0.1:5173/ 进入，管理端从 http://127.0.0.1:5173/admin 进入。所有后续页面必须可通过侧栏、移动抽屉或页面内操作到达。
