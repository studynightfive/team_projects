## 通用验收

- 用户主色 `#2B6CB0`，管理主色 `#553C9A`。
- 页面 `#F8FAFC`、卡片 `#FFFFFF`、表头/hover `#F1F5F9` 层级清楚。
- 无 emoji、纯黑、渐变、霓虹和彩色阴影。
- 卡片圆角 12px、按钮和输入框 8px；数字为 tabular-nums。
- 1920/1440/1280/375 无根级横向溢出、重叠或关键操作不可达。
- 移动端主要触控区域不小于 44px；抽屉焦点和关闭行为完整。
- 控制台错误 0、业务 `/api` 请求 0。

## 页面专项

### 登录

- 桌面品牌区宽度占比 55% ±0.5%，表单最大宽 360px，首屏无页面级滚动。
- 账号和密码空提交显示两个错误；密码显隐、记住我和辅助入口可键盘操作。
- 375px 不保留横向分屏，法律声明和全部入口可达。

### 用户工作台

- 六项导航、四张指标卡、四个知识库和四条团队动态存在。
- `Ctrl/Cmd + K` 聚焦搜索；侧栏 240px 可折叠为 64px。
- 375px 指标和内容区单列，底部四项导航可达。

### 管理中心

- 四张指标卡均有 Sparkline，每条七个数据点，共 28 点。
- 三项服务、三项治理事项和审计表存在；分页展示 disabled 状态。
- 审计表头计算颜色为 `rgb(241, 245, 249)`。
- 紫色只用于选中态、按钮、标签和趋势，不形成大面积内容背景。
