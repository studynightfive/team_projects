# M01 V2 资产清单

## 当前资产决定

- 产品名固定为“智能知识库平台”。
- 正式企业 Logo 尚未提供；当前使用 `@lucide/vue` 的 `BookOpenText` 线性图标组成可替换的平台标记，不宣称为真实企业 Logo。
- 业务图标来自 `@lucide/vue@1.24.0`，包许可证为 ISC，按需导入。
- 提示词指定的 `lucide-vue-next` 已由官方废弃，不能作为新依赖。
- 企业微信、钉钉、飞书官方品牌 SVG 未提供；登录页使用带中文可访问名称的通用线性图标，不冒充官方品牌资产。
- 不加载远程字体。字体栈优先本机 Inter、PingFang SC、Microsoft YaHei UI、Source Han Sans，并提供系统兜底。
- 不使用插画、照片、视频、Base64 图片、CDN 图标或远程 Web Font。

## 隐私规则

- Mock 姓名均为演示内容，不对应真实员工。
- 不使用真实头像、邮箱、账号、组织 ID、内部 IP、Token、Cookie 或下载地址。
- 审计 IP 使用 RFC 5737 文档保留地址。

## 后续替换

获得正式 Logo、品牌登录图标或本地字体资产后，必须记录来源、许可、文件哈希和使用范围，并重新生成受影响截图。禁止从竞品页面、搜索结果或不明素材包临时复制资产。
