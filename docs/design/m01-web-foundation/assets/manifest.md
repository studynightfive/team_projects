# M01 资产清单

## 决定

M01 不引入自定义 Logo、插画、图片、Web Font、视频或外部图标包。使用：

- 文字产品名“智能知识库平台”。
- 操作系统字体栈，定义于 [`../tokens.css`](../tokens.css)。
- 静态 artifact 使用无外部依赖的文本和 CSS 界面提示；M01 生产实现不为装饰图标新增依赖。确有可访问性或理解成本需要时，必须在唯一 P0 Issue 内先批准并精确锁定兼容图标包。

这不是临时占位。项目负责人通过对应 PR 的非作者批准后，它成为 M01 正式资产策略；未来获得品牌资产时必须通过新的 Issue/PR 更新来源、许可、版本和验收截图。

## 文件记录

| 文件 | 用途 | 来源 | 许可 | 当前状态 |
|---|---|---|---|---|
| 无 | M01 无自定义媒体资产 | 不适用 | 不适用 | 待项目负责人批准 PR |

## 禁止项

- 不从搜索引擎、竞品页面或生成工具临时取得 Logo/插画并称为真实资产。
- 不使用 Base64 图片、远程字体或 CDN 图标隐藏来源。
- 不把 PNG 验收图当作可复用页面资产。
- 不在截图中展示真实头像、真实姓名、邮箱或组织信息。
